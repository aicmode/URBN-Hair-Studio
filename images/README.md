# Image Replacement Guide

This project uses external editorial salon photos in `index.html` so the LP can be previewed immediately.

To replace them with local project assets, save files in this folder and update the corresponding `url(...)` or `src` values:

- `hero-salon.jpg` for the hero visual
- `style-01.jpg` through `style-08.jpg` for Featured Styles
- `stylist-rina.jpg`, `stylist-leo.jpg`, `stylist-maya.jpg` for Meet The Stylists
- `reservation-bg.jpg` for the reservation section

Recommended formats:

- Hero and reservation: 2200px wide, landscape
- Style cards and stylist photos: 900px wide, portrait, 4:5 crop
- Export JPG/WebP at 80-90 quality
